## Installation

Prerequisites: [Node.js](https://nodejs.org/) 6+, **npm** 3+.

    npm install --save @demvsystems/fuzzy-date

## Usage

### ES5

    var xxx = require('@demvsystems/fuzzy-date').default;

### ES2015+

    import xxx from '@demvsystems/fuzzy-date';
